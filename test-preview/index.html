Save-to-File Demo

This is a minimal test page for the File System Access API save-back feature.

Hover the top-left corner for ~400ms, or press E to enter edit mode. Edit any text, then press Ctrl/Cmd+S or click Save to file. The first save will ask you to pick this same index.html file to enable write-back.
