01 / OVERVIEW

不只是代码补全，
而是编程搭档

Claude Code 直接运行在你的终端中，深度理解项目上下文，能够自主执行从编写代码、修复 Bug 到重构架构的一切任务。无需复杂配置，开箱即用。

# 安装
$ npm install -g @anthropic-ai/claude-code

# 在任意项目目录中启动
$ cd your-project && claude
